In dieser Arbeit geht es um die Vorhersage von Autopreisen auf der Grundlage eines Datensatzes von Kaggle1. Wir werden verschiedene Modelle und Methoden unter Verwendung der Python-Bibliothek Sklearn2 und statmodels3 ausprobieren. Das Benchmark-Modell wird die lineare Regression sein, alle anderen Modelle werden mit diesem Modell verglichen.

Zum Ablauf: Zuerst werden die Daten importiert und bereinigt, dann werden die Daten in train-test-split Verfahren aufgeteilt. Der grosse Teil, train, wird zum Trainieren des Modells verwendet, während der kleine Teil, test, zur Überprüfung der Leistung verwendet wird.

df = (
    df
    .assign(
        manufacturer        = lambda df: df.manufacturer.astype('category'),
        model               = lambda df: df.model.astype('category'),
        year                = lambda df: pd.to_datetime(df.year, format='%Y'),
        mileage             = lambda df: df.mileage.astype('int'),
        engine              = lambda df: df.engine.astype('str'),
        transmission        = lambda df: df.transmission.astype('category'),
        drivetrain          = lambda df: df.drivetrain.astype('category'),
        fuel                = lambda df: df.fuel.astype('category'),
        mpg                 = lambda df: df.mpg.astype('float'),
        exterior_color      = lambda df: df.exterior_color.astype('category'),
        interior_color      = lambda df: df.interior_color.astype('category'),
        accidents_or_damage = lambda df: df.accidents_or_damage.astype('category'),
        one_owner           = lambda df: df.one_owner.astype('category'),
        personal_use_only   = lambda df: df.personal_use_only.astype('category'),
        seller_name         = lambda df: df.seller_name.astype('category'),
        seller_rating       = lambda df: df.seller_rating.fillna(0).astype('float'),
        driver_rating       = lambda df: df.driver_rating.fillna(0).astype('float'),
        driver_reviews_num  = lambda df: df.driver_reviews_num.fillna(0).astype('int'),
        price_drop          = lambda df: df.price_drop.fillna(0).astype('int'),
        price               = lambda df: df.price.astype('int'),
    )
)
